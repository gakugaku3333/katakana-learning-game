# 🌸 カタカナ学習ゲーム - Katakana Learning Game

子供向けの楽しい教育系Webアプリケーション。カタカナ、タイピング、算数を遊びながら学べます！

[🎮 ゲームをプレイする（GitHub Pages）](https://あなたのユーザー名.github.io/リポジトリ名/)

## ✨ 機能

### 📚 学習ゲーム
- **カタカナクイズ** - カタカナの読み方を学ぼう！全46文字に対応
- **小文字タイピング** - a-zの小文字タイピング練習（60秒）
- **大文字タイピング** - A-Zの大文字タイピング練習（60秒）
- **算数ゲーム** - たしざん、ひきざん、かけざん、わりざん

### 🎮 ミニゲーム（10種類以上）
- 🔨 もぐらたたき
- 🎈 ふうせんわり
- 🍎 フルーツキャッチ
- 🎨 いろあわせ
- 🃏 しんけいすいじゃく
- 👆 クリッククリック
- ⭐ ほしあつめ
- 🫧 バブルわり
- 🎯 えあわせ（マッチ3）
- 🖍️ おえかき

### 🏆 その他の機能
- **⭐ トークンシステム** - ゲームをクリアしてトークンをゲット！
- **🛍️ ショップ** - トークンでバッジ、アバター、エフェクトを購入
- **📊 統計** - 学習の進捗を確認
- **💾 自動セーブ** - 進捗はブラウザに自動保存

## 📱 対応デバイス

- 💻 デスクトップ（Chrome, Firefox, Safari, Edge）
- 📱 スマートフォン（iOS Safari, Android Chrome）
- 📱 タブレット（iPad, Android タブレット）

### iPadでの使い方
1. SafariブラウザでゲームのURLを開く
2. 「共有」ボタン → 「ホーム画面に追加」でアプリのようにアクセス可能！
3. タッチ操作でそのままプレイできます
